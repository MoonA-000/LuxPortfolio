# Random Quote Generator 🌟

A beautiful, modern, and interactive quote generator website that provides inspiring quotes from great minds across different categories.

## ✨ Features

- **🎲 Random Quote Generation**: Click the generate button to get a random inspirational quote
- **📂 Category Filtering**: Filter quotes by categories like Inspiration, Success, Wisdom, Love, and Life
- **📋 Copy to Clipboard**: Easily copy quotes with proper attribution
- **📤 Share Functionality**: Share quotes directly or copy to clipboard as fallback
- **📊 Statistics Tracking**: Keep track of quotes generated and favorites
- **🎨 Beautiful UI**: Modern design with smooth animations and gradients
- **📱 Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **⌨️ Keyboard Shortcuts**: Press spacebar to generate a new quote
- **🎉 Easter Eggs**: Hidden surprises for curious users
- **🏆 Milestone Celebrations**: Special notifications for quote generation milestones

## 🚀 How to Use

1. **Open the Website**: Simply open `index.html` in your web browser
2. **Generate Quotes**: Click the "Generate Quote" button or press the spacebar
3. **Filter by Category**: Use the category buttons to filter quotes by type
4. **Copy Quotes**: Click the copy button to copy the current quote to your clipboard
5. **Share Quotes**: Click the share button to share the quote (uses native sharing on supported devices)
6. **Track Progress**: Watch your statistics update as you generate more quotes

## 🎯 Categories Available

- **All**: Mix of all quote categories
- **Inspiration**: Motivational and inspiring quotes
- **Success**: Quotes about achievement and success
- **Wisdom**: Philosophical and wise sayings
- **Love**: Quotes about love and relationships

## 🔧 Customization

### Adding New Quotes
To add new quotes, edit the `quotes` array in `script.js`:

```javascript
{
    text: "Your quote here",
    author: "Author Name",
    category: "category-name"
}
```

### Styling Changes
Modify `styles.css` to customize:
- Colors and gradients
- Typography
- Animations
- Layout and spacing

### Adding Categories
1. Add new category buttons in `index.html`
2. Update the filter logic in `script.js`
3. Add quotes with the new category
