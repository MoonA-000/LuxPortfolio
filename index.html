Why is the download button disabled? Downloading YouTube videos is restricted by YouTube’s Terms of Service except where explicitly provided (for example, downloading your own uploads in YouTube Studio). Implementing a downloader also requires code beyond HTML (e.g., JavaScript or a server). This page remains strictly HTML and compliant.

How to get a copy legally

If you own the video (best option)

  1. Open YouTube Studio.
  2. Go to Content → find your video.
  3. Click the three‑dot menu (Options) → Download.

This is officially supported and preserves quality.

If you have permission from the owner

  • Ask the owner to share the original file (e.g., via Drive, Dropbox, or direct transfer).
  • Alternatively, they can temporarily grant you access to download in their YouTube Studio.

Make sure you have written permission when applicable.

Premium offline viewing (mobile)

  • With YouTube Premium, some videos can be saved for offline viewing in the app.
  • This is not a transferable file, but can satisfy travel/offline needs.

Optional: make an embed link

You can quickly build an embed URL to preview a video on any site that allows iframes. Replace the watch link like this:

https://www.youtube.com/watch?v=VIDEO_ID → https://www.youtube.com/embed/VIDEO_ID

Then place it inside:

<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>

This is for viewing only, not downloading.
